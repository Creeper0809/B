# v1 smoke tests (skeleton)

Each roadmap priority group (P0..P6) should have a minimal smoke test.

Recommended convention:
- `p0_syscall_ok.b`
- `p1_core_ok.b`
- `p2_io_ok.b`
- `p3_lex_ok.b`
- `p4_parse_ok.b`
- `p5_codegen_ok.b`
- `p6_e2e_ok.b`

Implementations will be added as v1 modules become real.
